# UML Diagrams

This folder contains PlantUML sources generated from the current project analysis.

Files:
- `class-diagram.puml` - core domain model and relations
- `use-case-diagram.puml` - main actors and system capabilities
- `sequence-reading-alert.puml` - MQTT reading ingestion and alert creation
- `sequence-auth-dashboard.puml` - login, token refresh, and dashboard access
- `activity-site-approval.puml` - site creation and approval workflow

Open these files in VS Code with a PlantUML extension to preview and export PNG/SVG.
