# Frontend Features (EmissionsIQ)

## Overview

- Dashboard overview with pollutant cards and mini-trend charts.
- KPI widgets for TD, EMJ, IPE, and RCO2.
- Latest alerts list with severity and timestamp.

## Pollutants and Readings

- Supports CO2, NOx, SO2, COV, PM25, temperature, humidity.
- Latest readings view with units per pollutant.
- Time-bucketed readings grouped by node and pollutant.

## History

- Historical charts by pollutant.
- Period filters (last hour, 24 hours, 7 days, 30 days, 12 months).

## Compliance

- Compliance page with KPI tracking and thresholds.
- Visual indicators for status by pollutant.

## Authentication

- Login flow with JWT session handling.
- Role-based access to protected pages.

## UX

- Responsive layout for desktop and mobile.
- Loading and empty-state messages.
